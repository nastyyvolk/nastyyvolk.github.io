FORMS

Form tags

Tag Description
<form> The wrapper element for all HTML forms.
<input> Creates a single input item.
<textarea> Creates a text-box for multi-line input.
<select> Creates a dropdown list.
<datalist> Creates a list of data that shown as a dropdown, although allows users to write alternative inputs as well.
<option> Creates an option in the dropdown list or datalist.
Note that inside <select> tag option needs a closing tag, but in <datalist> it is an empty tag.
<label> Creates a label for a form input.

Form attributes

Attribute Value Description
<form> action= "/file.html" Information filled in the form is passed to the source which is named in the action attribute.
<form> method= POST Used for long forms. May contain sensitive data. Adds or deletes information from a database.
<form > method= GET Used for short forms. No data should be added or deleted from a database.
required - Specifies that the field must include a value.
name Custom Used to save user's data in a database.
placeholder Custom Renders as a disappearing text inside the input area.
maxlength Custom Sets maximum length of the input.
minlength Custom Sets minimum length of the input.
pattern [A-Za-z]{3} (RegEx) Uses a regular expression to validate the value of the input, when the form is submitted.
